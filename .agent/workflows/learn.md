# /learn — Engineering Lessons

1. **Review** what was just completed (diff, PR, or conversation).

2. **Extract** lessons learned:
   - What worked well?
   - What was unexpected or tricky?
   - What patterns should be reused?
   - What anti-patterns were discovered?

3. **Document** in `temp/lessons/` as `YYYY-MM-DD-<topic>.md`.

4. **Update** relevant `.agent/skills/` if a pattern should be formalized.
